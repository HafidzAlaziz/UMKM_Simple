# Frontend-Only Deployment (Vercel)

Jika Anda hanya deploy frontend tanpa backend, aplikasi akan tetap berjalan dengan **mock data**.

## Apa yang Terjadi

- ✅ Frontend berjalan normal
- ❌ API calls akan gagal (backend tidak ada)
- ✅ Fallback ke mock/sample data
- ✅ UI tetap bisa dilihat dan dinavigasi

## Fitur yang Bekerja (Tanpa Backend)

### ✅ Bisa Diakses:
- Home page dengan sample products
- Product listing dengan mock data
- Product detail (jika ada fallback)
- Categories & Brands pages
- About page
- UI/UX showcase

### ❌ Tidak Berfungsi:
- Login/Register (butuh backend API)
- Add to cart (localStorage masih work)
- Checkout & Orders (butuh backend)
- Admin panel (butuh backend API)
- Real product data

## Solusi Lengkap: Deploy Backend

Untuk aplikasi fully functional, Anda perlu deploy backend juga.

### Recommended: Railway

1. **Buat akun di Railway.app**
   - https://railway.app
   - Login dengan GitHub

2. **Deploy Backend**
   - New Project → Deploy from GitHub
   - Pilih repository: `HafidzAlaziz/UMKM_Simple`
   - Root directory: `server`
   - Railway akan auto-detect Node.js

3. **Add PostgreSQL Database**
   - Add service → Database → PostgreSQL
   - Railway akan auto-create `DATABASE_URL`

4. **Set Environment Variables**
   ```
   DATABASE_URL=(auto-generated by Railway)
   JWT_SECRET=your_secret_here
   JWT_REFRESH_SECRET=your_refresh_secret_here
   NODE_ENV=production
   PORT=5000
   ```

5. **Deploy & Run Migrations**
   - Railway akan auto-deploy
   - Setelah deploy, run command di Railway:
   ```bash
   npx prisma migrate deploy
   npx prisma db seed
   npx ts-node scripts/create-admin.ts
   ```

6. **Update Frontend Environment**
   - Di Vercel, tambah environment variable:
   - `VITE_API_URL` = `https://your-railway-app.railway.app/api`
   - Redeploy frontend

### Alternative: Render.com

1. New Web Service
2. Connect GitHub
3. Root directory: `server`
4. Build: `npm install && npm run build`
5. Start: `npm start`
6. Add PostgreSQL database
7. Set environment variables
8. Deploy

## Quick Fix: Update API Base URL

Jika backend sudah deploy, update `client/src/lib/api.ts`:

```typescript
const api = axios.create({
    baseURL: import.meta.env.VITE_API_URL || 'http://localhost:5000/api',
    withCredentials: true,
});
```

Lalu set `VITE_API_URL` di Vercel environment variables.
